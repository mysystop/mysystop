# 错误提示
小米Node 4 上运行React-Native 应用，提示错误：
```
com.android.ddmlib.InstallException:Failed to establish session
```

#  解决方案

小米手机设置里-------开发者选项---------启用MIUI优化 关闭

完美解决！
